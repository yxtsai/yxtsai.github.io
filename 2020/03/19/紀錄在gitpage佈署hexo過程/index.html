但我還是不幸遇到問題，以下紀錄一下

我在deploy過程中，遇到了Permission denied

Error: Permission denied (publickey)

我在此因為沒設定 github SSH key，所以deploy失敗

解法：
SSH key產生教學

搞定SSH Key後，仍然失敗XDDD

解法：
到一開始init的folder中，刪除.deploy.git這個資料夾，接著

$ hexo g

$ hexo d

Folder

scaffolds : 文章的template，所有你寫的文章或者page都是基於這個架構上面


source : md檔案所在位置，要刪除文章、編輯文章都在這，source資料夾中絕對不能沒有任何md，否則Gitpage會404，所以至少發一篇文章之後再來刪除Quickstart那篇


themes : blog主題存放位置，google hexo themes 會有一大堆主題，可以任選，每天換一個？
注意修改主題時候，冒號後有一個半行空格，如果沒有是會失敗的

theme: theme_name

刪除文章

在source這個folder刪掉你不想要的文章（.md)後

$ hexo clean

$ hexo g

$ hexo d

編輯文章

Github page用markdown語法，我目前是用VScode在編寫，有關語法部分請參考
markdown語法
