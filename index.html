三月

Posted on 2020-03-21

「希望你平安健康」

每天都戴著口罩，噴著酒精，兩隻手稍有脫皮現象，即便是感染風險偏低的我，都有點耐不住，想逃離這裡，然後想到前線的醫療人員，還有我很在意的人，這些人面對這些卻無懼，非常尊敬也非常擔心，由衷希望這次台灣能撐過去。
搶購糧食、民生用品，假消息無孔不入，Y昨天跑去超市，傳了照片給我，架上的泡麵已經清空，我也有點擔心起來，問他要不要早點回來，一起去好市多採購一些東西，確保存糧，說是存糧，大概也是一些垃圾食物。

但Y是這樣的人，天不怕地不怕，就怕麻煩？說了不用，因為我前幾天已經問過一次，所以對於這樣的答案也不太意外，但他隔天問我什麼時候要去好市多？我嘲笑了他一番，問他要去買什麼？他給我的回答是他要買酪梨，唉，完全搞不懂。

要念的書很多，計組老師每週都有一個影片，每週一考小考，我都懷疑自己是不是看錯影片或者拿錯考卷，週三會作一個整理，週三老師總會說下週考這個，目前為止考出來紀錄是0次，我真的會完蛋吧。

結尾：我要去唸計組了ㄅㄅ，應該不會有人找到這個blogㄅ(^u^)。

但我還是不幸遇到問題，以下紀錄一下

我在deploy過程中，遇到了Permission denied

Error: Permission denied (publickey)

我在此因為沒設定 github SSH key，所以deploy失敗

解法：
SSH key產生教學

搞定SSH Key後，仍然失敗XDDD

解法：
到一開始init的folder中，刪除.deploy.git這個資料夾，接著

$ hexo g

$ hexo d

Folder

scaffolds : 文章的template，所有你寫的文章或者page都是基於這個架構上面


source : md檔案所在位置，要刪除文章、編輯文章都在這，source資料夾中絕對不能沒有任何md，否則Gitpage會404，所以至少發一篇文章之後再來刪除Quickstart那篇


themes : blog主題存放位置，google hexo themes 會有一大堆主題，可以任選，每天換一個？
注意修改主題時候，冒號後有一個半行空格，如果沒有是會失敗的

theme: theme_name

刪除文章

在source這個folder刪掉你不想要的文章（.md)後

$ hexo clean

$ hexo g

$ hexo d

編輯文章

Github page用markdown語法，我目前是用VScode在編寫，有關語法部分請參考
markdown語法

這裡推薦兩個vscode的extension for markdown

markdown all in one

markdown preview enhanced

發布文章失敗？

有時候deploy後沒能即時看到更新後的頁面，刪除cookie以及換個裝置看看，只要md沒問題以及deploy成功的話。
