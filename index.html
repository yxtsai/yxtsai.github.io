四月

Posted on 2020-04-03 Edited on 2020-04-11

愛麗絲1913
回到家鄉時，若是霧茫茫以及有點冷冽的早晨，會讓我時常想起A，曾跟Y說了A的故事，我想不起來當時Y說了什麼，Y從未因為相關事跟我吵架，Y不是一個溫柔的人，即便這樣他仍願意接住了我所有不確定。
A的樣子在我腦海中逐漸模糊，我回想著，不過記憶很神奇，有時候回憶是會騙人的，所以也許我記憶中的A也不全為那樣，但沒辦法證實了，A真的是一個非常特別的存在，直至現在。

我在大二那年過得非常痛苦，很常coding到哭出來，在沒有歸屬感以及作業寫不完的情況又無法求助，那時候每天去學校幾乎是折磨，後來撐不住了，打電話跟家母說要休學，意外的家母說了好，而非逼迫我繼續念。

現在問我會不會後悔休學？也許會，我在那學期修了一堆很棒的課，只無奈當時情況真的超出我負荷範圍，我好幾次有縱身往下跳的想法，當時如果選擇繼續修課，也有可能我就不在世界上了。我曾想到A說的話，活在世界上本身就是一件很難的事情，值得感謝，我非常感謝A，如果有一個人可以永遠幸福快樂生活，那我希望那個人會是A。

三月

Posted on 2020-03-21 Edited on 2020-04-07

一、「希望你平安健康」

每天都戴著口罩，噴著酒精，兩隻手稍有脫皮現象，即便是感染風險偏低的我，都有點耐不住，想逃離這裡，然後想到前線的醫療人員，還有我很在意的人，這些人面對這些卻無懼，非常尊敬也非常擔心，由衷希望這次台灣能撐過去。
搶購糧食、民生用品，假消息無孔不入，Y昨天跑去超市，傳了照片給我，架上的泡麵已經清空，我也有點擔心起來，問他要不要早點回來，一起去好市多採購一些東西，確保存糧，說是存糧，大概也是一些垃圾食物。

但Y是這樣的人，天不怕地不怕，就怕麻煩？說了不用，因為我前幾天已經問過一次，所以對於這樣的答案也不太意外，但他隔天問我什麼時候要去好市多？我嘲笑了他一番，問他要去買什麼？他給我的回答是他要買酪梨，唉，完全搞不懂。

要念的書很多，計組老師每週都有一個影片，每週一考小考，我都懷疑自己是不是看錯影片或者拿錯考卷，週三會作一個整理，週三老師總會說下週考這個，目前為止考出來紀錄是0次，我真的會完蛋吧。

結尾：我要去唸計組了ㄅㄅ，應該不會有人找到這個blogㄅ(^u^)。

在夢境裡，他們在一起了，醒來時會想明明日常生活沒有交集的兩人，卻不間斷的夢到，在夢境裡總是笑著的人，醒來的現實於我為惡夢，我甚至因半個月都在思考這事情而求助了心輔，就寄望能夠與傷痛共處，如果哪天能夠在醒來中平靜的繼續生活就好啦。

三、月末

草東沒有派對-我不知道你知不知道欸
壓力大的時候就容易做夢，頻繁看3C容易睡不好。從開始上九個小時課的那天開始，坐在教室中，看著一堆要跟你競爭的人，壓力不由得大了起來，也許是這樣才致使每天在不同惡夢中度過，且醒來後猶有夢還沒結束之感，但很多時候都想不起來到底做了什麼夢。

Y在交往初期說我很會說夢話，交往久了，Y已經被訓練成即使我說夢話他還是呼呼大睡，倒是室友會說我前一天又說什麼荒謬的夢話，比方說我曾說了喵喵，或者畢恭畢敬的說謝謝，太有禮貌了吧。

武漢肺炎疫情越來越嚴重，從不到一百例到現在突破兩百例，且有未知感染源的確診者，情緒亦趨浮躁，希望所有課都能早日遠端授課，如果需要回家避難的話，想把Y帶上，順便實施小廚娘計畫，最近自己下廚頻率變高了，胡椒雞湯、香菇雞湯，在玩德撲那天，繞去了市場，買了十斤青梅準備來作脆梅，夏天到了。

解法：
到一開始init的folder中，刪除.deploy.git這個資料夾，接著

$ hexo g

$ hexo d

Folder

scaffolds : 文章的template，所有你寫的文章或者page都是基於這個架構上面


source : md檔案所在位置，要刪除文章、編輯文章都在這，source資料夾中絕對不能沒有任何md，否則Gitpage會404，所以至少發一篇文章之後再來刪除Quickstart那篇


themes : blog主題存放位置，google hexo themes 會有一大堆主題，可以任選，每天換一個？
注意修改主題時候，冒號後有一個半行空格，如果沒有是會失敗的

theme: theme_name

刪除文章

在source這個folder刪掉你不想要的文章（.md)後

$ hexo clean

$ hexo g

$ hexo d

編輯文章

Github page用markdown語法，我目前是用VScode在編寫，有關語法部分請參考
